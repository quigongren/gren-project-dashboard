# Gren Project Dashboard

This repository supports documentation and visualization for the Jill Gren Health Management Project and the Gren Family Lineage and Genealogy research.

## Directory Structure

````
documents/
├── deliverables/    # Final reports, narrated scripts, documentation
├── drafts/          # Works in progress

assets/
├── audio/           # Narration files
├── images/          # Project visuals (portraits, diagrams, maps)

project-management/  # Tracking tools, Gantt charts, updates
visuals/              # Infographics, rendered visuals, illustrations
````

## Purpose
To maintain a versioned, accessible archive of key documents, multimedia assets, and structured data for ongoing genealogy and historical research related to Jill Gren, Katherine Fischer, and connected families.

## Maintainers
Project coordinated by AI-assisted archival and historical analysis (ChatGPT + Drive + GitHub stack).
