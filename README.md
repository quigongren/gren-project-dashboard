# Gren Project Dashboard

This repository serves as the central dashboard and documentation hub for the Gren Family Lineage and Genealogy initiative.

## Folder Structure

- `documents/drafts/`: Working documents and report drafts.
- `assets/images/`: Visual assets including diagrams, family trees, and archival photos.
- `visuals/`: Finalized infographics, data visualizations, and maps.
- `project-management/`: Planning materials, timelines, kanban exports, and process notes.

## Purpose
This GitHub repo is intended to mirror and synchronize key parts of the Google Drive project folder for traceability, versioning, and team-wide transparency.

---

All collaborators are encouraged to contribute updates and additions to these folders.
